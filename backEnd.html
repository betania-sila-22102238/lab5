Back-End

O back-end é a parte da aplicação web que fica no servidor e é responsável pelo processamento e armazenamento dos dados. As principais tecnologias utilizadas no desenvolvimento de sistemas back-end são:

  • PHP
  • Python
  • Java
  • .NET
  • Node.js
  • Ruby on Rails
  • SQL
